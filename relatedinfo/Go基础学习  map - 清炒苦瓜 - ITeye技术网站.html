Go基础学习: map

    博客分类：
  • golang
gogolang 
记录下Go的基础学习笔记，仅供自己参考，如果能帮到您，那是我的荣幸

map，Go里面称之为映射，一种像脚本语言使用数组的方式
常规用法示例:
Go代码  
  1. m := map[string] int {  
  2.     "aa":100,  
  3.     "bb":100,  
  4.     "cc":100,  
  5. }  
  6. fmt.Printf("%d--%d\n", len(m),m["aa"]);  
	m := map[string] int {
		"aa":100,
		"bb":100,
		"cc":100,
	}
	fmt.Printf("%d--%d\n", len(m),m["aa"]);

输出:
3--100

要注意的是，如果取一个不存在的索引，会返回对应类型零值:
Go代码  
  1. m := map[string] int {  
  2.     "aa":100,  
  3.     "bb":100,  
  4.     "cc":100,  
  5. }  
  6. fmt.Printf("%d\n", m["xx"]);//不存在，该map对应的类型是int，所以返回int类型的零值:0  
	m := map[string] int {
		"aa":100,
		"bb":100,
		"cc":100,
	}
	fmt.Printf("%d\n", m["xx"]);//不存在，该map对应的类型是int，所以返回int类型的零值:0

输出:
0

你可能就要问了，我们日常开发中，经常要检测有没有这个索引时，如何做啊？总不能判为0时，就表示不存在吧，因为有可能我的值就是0啊，放心，GO都想到了，这样即可:
Go代码  
  1. m := map[string] int {  
  2.     "aa":0,  
  3.     "bb":100,  
  4.     "cc":100,  
  5. }  
  6. n, ok := m["aa"]  
  7. if ok {  
  8.     fmt.Printf("found aa:%d \n",n);  
  9. }else{  
  10.     fmt.Printf("not found aa \n");  
  11. }  
  12. n, ok = m["xx"]  
  13. if ok {  
  14.     fmt.Printf("found xx:%d \n",n);  
  15. }else{  
  16.     fmt.Printf("not found xx \n");  
  17. }  
	m := map[string] int {
		"aa":0,
		"bb":100,
		"cc":100,
	}
	n, ok := m["aa"]
	if ok {
		fmt.Printf("found aa:%d \n",n);
	}else{
		fmt.Printf("not found aa \n");
	}
	n, ok = m["xx"]
	if ok {
		fmt.Printf("found xx:%d \n",n);
	}else{
		fmt.Printf("not found xx \n");
	}

输出:
found aa:0
not found xx

还有一种情况，我只想判断有没有存在，而不想取里面的值，可以吗？
可以的，这样即可:
Go代码  
  1. m := map[string] int {  
  2.     "aa":0,  
  3.     "bb":100,  
  4.     "cc":100,  
  5. }  
  6. _, ok := m["aa"]//使用_表示一个空白标识，可以理解为放弃这个返回值  
  7. if ok {  
  8.     fmt.Printf("found aa\n");  
  9. }else{  
  10.     fmt.Printf("not found aa \n");  
  11. }  
  12. _, ok = m["xx"]  
  13. if ok {  
  14.     fmt.Printf("found xx\n");  
  15. }else{  
  16.     fmt.Printf("not found xx \n");  
  17. }  
	m := map[string] int {
		"aa":0,
		"bb":100,
		"cc":100,
	}
	_, ok := m["aa"]//使用_表示一个空白标识，可以理解为放弃这个返回值
	if ok {
		fmt.Printf("found aa\n");
	}else{
		fmt.Printf("not found aa \n");
	}
	_, ok = m["xx"]
	if ok {
		fmt.Printf("found xx\n");
	}else{
		fmt.Printf("not found xx \n");
	}

输出:
found aa
not found xx

定义完map后，还可以加值吗？可以的:
Go代码  
  1. m := map[string] int {  
  2.     "aa":0,  
  3.     "bb":100,  
  4.     "cc":100,  
  5. }  
  6. m["xx"] = 111  
  7. fmt.Printf("%d\n", m["xx"]);  
	m := map[string] int {
		"aa":0,
		"bb":100,
		"cc":100,
	}
	m["xx"] = 111
	fmt.Printf("%d\n", m["xx"]);

输出:
111

如何删除里map里面的某个索引及对应的值:
Go代码  
  1. m := map[string] int {  
  2.     "aa":0,  
  3.     "bb":100,  
  4.     "cc":100,  
  5. }  
  6. m["xx"] = 111  
  7. delete(m,"xx")  
  8. fmt.Printf("%d\n", m["xx"]);  
	m := map[string] int {
		"aa":0,
		"bb":100,
		"cc":100,
	}
	m["xx"] = 111
	delete(m,"xx")
	fmt.Printf("%d\n", m["xx"]);

输出:
0

如何遍历map:
Go代码  
  1. m := map[string] int {  
  2.     "bb":100,  
  3.     "cc":100,  
  4. }  
  5. m["xx"] = 111  
  6. for k,v := range m{  
  7.     fmt.Printf("%s=%d\n",k,v);  
  8. }  
	m := map[string] int {
		"bb":100,
		"cc":100,
	}
	m["xx"] = 111
	for k,v := range m{
		fmt.Printf("%s=%d\n",k,v);
	}

输出:
bb=100
cc=100
xx=111
